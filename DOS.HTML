¿Qué es el HTML? ¿En qué consiste?

HTML no es un lenguaje de programación; es un lenguaje de marcado que define la estructura de tu contenido. HTML consiste en una serie de elementos que se usa para encerrar diferentes partes del contenido para que se vean o comporten de una determinada manera. Es el código que se utiliza para estructurar y desplegar una página web y sus contenidos.
Es un conjunto de elementos, etiquetas y atributos

¿Qué es una etiqueta? ¿Funciona si se escribe tanto en mayúscula como en minúscula? ¿Cuál es la mejor práctica para escribir etiquetas?

Las etiquetas son bloques de códigos que dan funcionalidad y estructura al contenido de una página web. Estos bloques sirven para que un navegador navegador muestre de forma adecuada la información.
Da igual si se utiliza mayúscula o minúsculas para escribir las etiquetas, pero por convención se usa minúscula.
Una buena manera de tener una mejor practica para escribir etiquetas es escribir en minusculas, dar formato al documento, separar el contenido de la presentacion, etc.

¿A qué llamamos elemento HTML? ¿Cuál es su anatomía? Explicar sus partes.

Un elemento HTML consta de una etiqueta de apertura, contenido (si lo tiene) y una etiqueta de cierre.Un elemento es parte de una página web.
La etiqueta de apertura:consiste en el nombre del elemento,encerrado por paréntesis (<>) de apertura y cierre. Establece dónde comienza
La etiqueta de cierre: es igual que la etiqueta de apertura, excepto que incluye una barra de cierre (/) antes del nombre de la etiqueta. Establece dónde termina el elemento.
El contenido: este es el contenido del elemento.
El elemento:la etiqueta de apertura, más la etiqueta de cierre, más el contenido equivale al elemento.

Averigua que es GIT y GITHUB

GIT es un sistema de control de versiones distribuido, un clon local del proyecto es un repositorio de control de versiones completo. Estos repositorios locales plenamente funcionales permiten trabajar sin conexion o de forma remota
GITHUB es una plataforma para alojar proyectos utilizando el sistema de control de versiones GIT. Aloja el codigo de las aplicaciones de cualquier desarrollador web.
