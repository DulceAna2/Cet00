¿Cuáles son los tipos de títulos que hay?

Los elementos de encabezado implementan seis niveles de encabezado del documento, estos son : h1, h2, h3, h4, h5y h6

¿Cuál es la importancia del número que lleva la etiqueta de título?

La importancia del numero que lleva la etiqueta de titulo es su tamaño. Cuando el numero sea mas chico el titulo va a ser mas grande y mientras mas grande sea el numero mas chico va a ser el titulo. "h1" es el más importante, y "h6", el menos importante.

¿Qué es lo que debe evitarse para que el uso de los títulos sea accesible a las personas no videntes?. Justificar

Las personas no videntes utilizan una salida de texto a voz. Para que no haya una confusion, los titulos principales deben ser mas grandes que los titulos secundarios.Es importantes que se resalte los titulos principales, organizando los niveles para no confundir al usuario

¿Qué es un atributo en una etiqueta HTML? Dar dos ejemplos de etiquetas HTML con atributos, marcando sus respectivos atributos.

Los atributos en una etiqueta HTML son valores adicionales que configuran los elementos o ajustan su comportamiento de diversas formas para cumplir los criterios del usuaio.

Ej, "(html lang="en")" donde, html es la etiqueta y lang es el atributo, lang marca el lenguaje del contenido de la pagina. Otro ejemplo, "(meta charset="UTF-8")", meta es la etiqueta y charset es el atributo, el charset es para representar un lenguaje o el alfabeto. Para que sea algo mas mundial usamos el "UTF-8"

¿Qué hace la etiqueta "p"? ¿Cuáles son sus atributos?

La etiqueta "p" sirve para dividir el texto en parrafo

Los atributos de la etiqueta p son: title, style, id, class, dir, lang,

title: título consultivo del elemento.

style: información de estilo en línea.

class: identificador a nivel de documento

dir: dirección del texto débil/neutral

id: identificador único a nivel de documento.

lang:información sobre el idioma

Investiga qué hacen y cómo se utilizan las siguientes etiquetas HTML:Strong, i,u,sub,sup,em,a,span

La etiqueta strong: sirve para marcar con especial énfasis las partes más importantes de un texto.

La etiqueta i: sirve para mostrar el texto con un estilo en cursiva o italica

La etiqueta sub: sirve para representar texto en formato de subíndice dentro de un documento HTML

La etiqueta sup: es casi igual que la etiqueta sub. La etiqueta sup es mas chica y funciona como un superindice

La etiqueta em: sirve para marcar con énfasis las partes importantes de un texto.

La etiqueta a: sirve para crear un enlace a otras páginas de internet, archivos o ubicaciones dentro de la misma página, direcciones de correo, o cualquier otra URL.

La etiqueta span: funciona como un contenedor en línea. Sirve para aplicar estilo al texto o agrupar elementos en línea.
